Frontend for smart home devices on my home network.
Features : 
  - Zoneminder (cameras)
    - View all cameras
    - List and view last 100 events
  - Philips Hue
    - View all lights
    - View all groups
    - Switch lights/groups on and off
    - Set brightness of individual lights
    - Configurable refresh rate
